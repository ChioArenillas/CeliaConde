# Celia Conde - Psychology Website

## 🧩 Description

Celia Conde is a responsive React-based web application developed for a professional psychologist. The project focuses on providing a clean, accessible and conversion-oriented interface for potential clients.

It includes a fully structured SPA with routing between pages, a contact system integrated with email sending functionality, and multiple conversion points such as CTA sections and direct WhatsApp/Instagram links. The project also includes legal pages to comply with professional web requirements.

The goal of the project is to combine UX clarity, performance, and professional presentation in a real client-oriented website.

## 🚀 Live Demo

https://celia-conde.vercel.app/

## 🚀 Technologies

* HTML5
* CSS3
* TypeScript
* React
* React Router
* EmailJS

## 🎯 Features

* Responsive multi-page SPA built with React Router
* Contact form with real email sending integration (EmailJS)
* CTA sections designed for conversion (WhatsApp / contact flow)
* Integration of WhatsApp and Instagram links
* Legal pages (Privacy Policy, Cookies, Legal Notice)
* Fully responsive UI adapted to mobile, tablet and desktop

## 🧠 Key Concepts

* Component-based architecture in React
* Client-side routing with React Router
* Form handling and asynchronous email sending
* Responsive design with CSS variables and media queries
* UX-focused layout for conversion optimization
* Separation of pages and reusable components
